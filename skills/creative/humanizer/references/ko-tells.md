# 한국어 AI 글쓰기 특징 (KO Tells)

한국어 텍스트 감사 시 이 목록과 대조한다. 패턴 → 왜 AI 티가 나는가 → 고치는 법.

## 어미·종결 (Endings)

1. **"~할 수 있습니다" 반복** — 가능 표현 남발. 단언으로 바꿔라. "처리할 수 있습니다" → "처리한다".
2. **"~라고 할 수 있습니다" / "~인 셈입니다"** — 헤징. 삭제하고 직접 말하라.
3. **"~하는 것이 좋습니다" 반복** — 권유체 강박. "~하세요" 또는 단언.
4. **과도하게 일관된 정중체** — 모든 문장이 "~습니다"로 끝나 리듬이 없다. 길이와 어미를 섞어라.
5. **"~되어 있습니다" 수동 남용** — "설계되어 있습니다" → "설계했다" (행위자 명시).

## 어휘 (Lexical)

6. **번역체 클리셰**: 원활한, 강력한, 혁신적인, 손쉽게, 효율적으로, 최적화된, 매끄러운 → 구체어로.
7. **"당신은" 남용** — 한국어는 주어 생략이 자연스럽다. 대부분 빼라.
8. **"~에 대해 알아보겠습니다 / 살펴보겠습니다"** — 도입 상투구. 삭제.
9. **"중요한 것은 / 주목할 점은 / 핵심은"** — 강조 상투구 남발. 그냥 말하라.
10. **명사화 과다**: "최적화의 구현을 통해" → "최적화해서".

## 전환·접속 (Transitions)

11. **"또한 / 더불어 / 나아가 / 게다가"** 기계적 나열 → 줄이고 흐름으로 연결.
12. **"뿐만 아니라 ~도"** 강박 (영어 not only...but also 의 번역).
13. **"하지만 반면에 / 그러나 한편"** — 균형 강박. 입장이 필요하면 입장을 취하라.

## 구조 (Structural)

14. **3단 병렬 강박** — "빠르고, 강력하고, 직관적인". 둘 또는 넷으로, 또는 풀어써라.
15. **균일한 문장 길이** — AI는 중간 길이를 고르게 쓴다. 짧게 끊어라. 한 단어 문장도 OK.
16. **불릿 강박** — 흐르는 글을 목록으로. 정말 필요할 때만 목록.
17. **볼드 + 이모지 남발** (**핵심** 🔥, **결론** ✅) → 제거. 본문은 plain text.

## 결론·도입 (Framing)

18. **"결론적으로 / 요약하자면 / 정리하면"** — 삭제, 마지막 문장이 스스로 결론이 되게.
19. **"오늘날 빠르게 변화하는"** / "급변하는 시대에" → 삭제.
20. **"~는 필수입니다 / 더 이상 선택이 아닙니다"** — 과장 단언. 구체적 근거로.

## 톤 (Tone)

21. **빈 열정**: "놀라운, 혁신적인, 강력한" 구체성 없이 → 사실로.
22. **과도한 헤징 누적**: "~수도 있습니다", "경우에 따라", "일반적으로" 중첩 → 직접.
23. **사대주의 도입**: "좋은 질문입니다!", "물론입니다!" → 삭제.

## 빠른 체크 (스캔 순서)

"~할 수 있습니다" 빈도 → "또한/더불어" 빈도 → 3단 병렬 → "결론적으로" → 볼드/이모지 → 균일 문장 길이 → "당신은".
